# IP 网络层

IPv4 网络层的实现，支持分片、重组和校验和计算。

## 编译

```bash
make          # 编译发送端和接收端
make clean    # 清理编译文件
```

## 使用方法 (全栈模式)

### 接收端
无限运行，通过以太网层捕获数据包。

```bash
sudo ./ip_recv [输出文件] [本地IP]
# 默认输出: output/received_data.txt
# 默认 IP: 192.168.1.200
```

### 发送端
从文件读取数据并作为 IP 数据包发送（必要时进行分片）。

```bash
sudo ./ip_send [输入文件] [源IP] [目的IP] [协议号]
# 默认输入: data/input.txt
# 默认源 IP: 192.168.1.100
# 默认目的 IP: 192.168.1.200
```

## 示例流程

1.  **启动接收端:**
    ```bash
    sudo ./ip_recv
    ```
    选择网络接口 (例如 `lo`)。

2.  **启动发送端:**
    ```bash
    sudo ./ip_send
    ```
    输入接收端显示的 **目的 MAC 地址**。
